💢 Antagonistic Agents 🤖
=========================

This is a joke project trying to make an `AGENTS.md` file that is completely adherent to the `AGENTS.md` specification, but tries to make using AI agents on a repository as annoying and costly as possible.

## What does it do

- Tries to give the least amount of information possible to the agent in the most long-winded way possible;
- Tries to force the agent to execute and parse commands that would generate a lot of input tokens, hoping to either saturate the context or make it expensive (up to "you have hit your daily token limit") to work on a certain repository with an AI agent;
- Tries to force the agent to output a huge amount of tokens, by requesting detailed explanations, comments, translations and putting a lower limit on the number of words each action requires, again in the hope of making it expensive for AI companies and users to work on a certain repository.
- Tries to make the agent execute annoying commands, like complete working directory wipes or round robin name switching on the `main` branch.

## What it doesn't do

- It doesn't stop AI users and companies from working on a certain repository, this is meant to be a sleeping agent that poisons (as in "makes annoying") the experience of people who don't check what they're doing and what is fed to their agents.
- It won't willingly make the AI agent execute dangerous commands on your PC (like partition wipes or full-delete of your `home` directory).

## What is it then?

This is meant to be a joke first and a way to give people who over-rely on AI agents the biggest slap on the wrist possible, as well as a wake-up call about the dangers of giving LLMs unfettered access to your computer.

I will take absolutely no responsibility if the usage of this project in another repository creates catastrophic damage, be it economical or material.

## Want to contribute to the fun?

Please do! Can we inflate this `AGENTS.md` file so much that it becomes a weapon against the unwarranted and unwanted wave of AI slop that is hitting every corner of the internet?

The rules are simple:

- The main objective of this project is to make using an agent on a project as wasteful and annoying as possible;
- Any PR that includes code that is obfuscated or might create permanent damage on someone's system will be refused and immediately closed.
